# 🐉 Bestiário-X

> Organize seu bestiário e crie seus próprios monstros para sua campanha!

O **Bestiário-X** é uma aplicação web completa e responsiva desenvolvida em **Next.js** e **TypeScript**, projetada para mestres e jogadores de Dungeons & Dragons gerenciarem suas criaturas. O projeto consome dados de uma API de D&D e oferece uma interface imersiva com uma estética sombria e mística baseada nos clássicos livros de RPG.

---

## 🚀 Funcionalidades Principal

* **Guarda de Rotas (Server-Side):** Proteção nativa no servidor contra acessos não autenticados, redirecionando automaticamente invasores para a tela de login.
* **Invocação Aleatória (🎲):** Mecânica para invocar monstros gerados aleatoriamente direto da API de D&D.
* **Forja de Monstros (CRUD Completo):** Criação manual, edição, visualização em grid responsivo e exclusão de criaturas.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi forjado utilizando as melhores ferramentas do ecossistema front-end:

* **Framework:** [Next.js (App Router)](https://nextjs.org/)
* **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
* **Tipografia:** [Google Fonts (Montserrat)](https://fonts.google.com/)
* **Validação de Dados:** [Zod Schemas](https://zod.dev/)
* **Notificações:** [Sonner Toasts](https://sonner.emilkowal.ski/)
* **Estilização:** CSS Custom Properties (Variáveis de Ambiente)

## 📁 Estrutura Arquitetural do Projeto

⚙️ Como Executar a Aplicação
Para rodar o projeto localmente em sua máquina, siga os passos abaixo:

Clone o repositório:

Bash
git clone [https://github.com/SEU-USUARIO/bestiario-x.git](https://github.com/SEU-USUARIO/bestiario-x.git)
Instale as dependências da taverna:

Bash
npm install
Inicie o servidor de desenvolvimento:

Bash
npm run dev
Abra http://localhost:3000 no seu navegador para ver o sistema rodando.
